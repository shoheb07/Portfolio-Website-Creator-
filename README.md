# Portfolio-Website-Creator-
Portfolio Website Creator in Python can be built as a CLI tool that generates a complete personal portfolio (HTML/CSS) from user input
portfolio_creator/
│── main.py
│── templates/
│     ├── index_template.html
│     └── style.css
│── output/
│     └── index.html
│── README.md
│── requirements.txt

Features
Generate portfolio website automatically
Custom name, bio, skills, projects, contact info
Clean modern UI (HTML + CSS)
No coding required for users
Saves output as ready-to-host website

Future Improvements
Add Flask web interface
Add themes
Add image upload
Deploy to GitHub Pages automatically
